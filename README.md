# Encryption Demo

This program demonstrates symmetric and asymmetric encryption in Python.

- Symmetric encryption uses AES (Fernet) with one key.
- Asymmetric encryption uses RSA with a public and private key.

The program:
1. Takes a message as input
2. Encrypts it
3. Decrypts it
4. Prints keys, input, encrypted output, and decrypted output

## How to Run
pip install cryptography
python main.py
